3) WINDOWS

Logon skript je spouštěn při přihlášení pomocí Task Scheduleru.

Skript logonCheck.ps1 kontroluje aktuálnost systémových aktualizací a antivirového programu a jejich aktuální běh, a při nalezených nedostatcích uživatele informuje.

Do newUser.ps1 jsem pro vytvoření scheduled task přidal do místa, kde se uživatel vytváří, řádky:

$LOGON_SCRIPT = "C:\...\logonCheck.ps1" # path to the logon script

# Assign Logon script
    $Action = New-ScheduledTaskAction -Execute 'PowerShell.exe' -Argument "-ExecutionPolicy Bypass -File `"$LOGON_SCRIPT`""
    $Trigger = New-ScheduledTaskTrigger -AtLogOn
    Register-ScheduledTask -Action $Action -Trigger $Trigger -TaskName "LogonCheck" -User "$name" -RunLevel Highest
